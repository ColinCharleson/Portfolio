Neolocity

Neolocity is a 3D open-world action-adventure game where players use parkour and combat skills to complete missions in a dystopian city. Built in Unity, it features mechanics like wall-running, gliding, and combat using a weaponized umbrella.

Year: 2023
Role: Game Designer, Level Designer, Programmer
Team Size: 5
Development Time: 7 Months
Engine: Unity

Itch Page

Major Contributions

  • Designed and tuned movement options: Gliding, Jump Boost, Wall Running
  • Combat system with melee attack, blocking and deflecting attacks
  • Designed open-world map
  • Added skills and progression system
  • Implemented dynamic dialogue based on player progress
  • Modular Mission System with 5 different mission types
  • Designed and implemented the UI
  • Accessibility settings
